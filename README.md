# Dashboard Facelift Backup Project

Using Vue3 + Vite, Node v16, yarn



## Project Setup

```sh
yarn
```

### Compile and Hot-Reload for Development

```sh
yarn dev
```

### Type-Check, Compile and Minify for Production

```sh
yarn build
```

### Take a look at production build

```sh
yarn try
```

### Factory Reset

```sh
yarn clean
```
